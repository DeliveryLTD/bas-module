# Support

Official releases of **Delivery Connect for BAS** are maintained by the Delivery Auto IT team.

## Support channels

- **Website:** https://delivery-auto.com.ua
- **GitHub Issues:** bug reports and feature requests
- **Email:** bas-module@delivery-auto.com.ua

## Scope

| | |
|---|---|
| **Official releases** | Supported by the Delivery Auto IT team. |
| **Customer-modified forks** | Supported on a **best-effort** basis. |

## Please do not publish secrets

Do not include API keys, passwords, personal data, phone numbers, addresses,
real waybill numbers, screenshots with credentials, or production logs in
public issues. See [SECURITY.md](SECURITY.md) for details.
