# Contributing

We welcome bug reports and improvement suggestions.

## Before submitting an issue

- Check the [latest release](https://github.com/DeliveryLTD/bas-module/releases/latest).
- Specify your **BAS / BAF / 1C platform** version.
- Specify the **configuration** name and version.
- Specify whether **regular** or **managed** forms are used.
- Specify the **module** version.
- **Remove** API keys, passwords, personal data and real waybill numbers.

## Pull requests

Pull requests are reviewed by the Delivery Auto IT team.

See also [SUPPORT.md](SUPPORT.md) and [SECURITY.md](SECURITY.md).
